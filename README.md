## Aplikace chemie.
### Zatím ve vývoji, může se chovat nestabilně
<p>Původní verze <a href="https://github.com/Johan5140/ChemieApp">zde</a></br>
Aplikace je pouze v 🇨🇿 jazyce. App is only in Czech language</br>
